Overview

Recycle Go is an innovative circular packaging system designed to address Taiwan's e-commerce packaging waste problem. This project proposes integrating ARM recycling machines with PackAge+ reusable packaging systems by establishing convenient return points on university campuses, combined with digital incentive mechanisms to increase packaging return rates and reduce overall circulation costs.

Research Methodology

User Research

  • In-depth Interviews: 10+ university student user interviews
  • Survey Research: Packaging usage behavior and recycling motivation analysis
  • Behavioral Observation: Current recycling system usage pattern analysis

Market Analysis

  • Competitive Analysis: In-depth comparison of 5+ circular packaging models
  • Policy Research: Taiwan EPA related policy analysis
  • Technical Assessment: ARM system and PackAge+ integration feasibility

Problem Statement

Current Challenges

  • Low return rates: Taiwan's e-commerce packaging return rate is only about 25%, with large amounts of reusable packaging being discarded
  • Uneven geographic distribution: Return points concentrated in urban areas, low participation in remote regions
  • Insufficient incentive mechanisms: Lack of effective motivation to drive consumer participation in recycling
  • Convenience issues: Complex existing return processes increase barriers to user participation

Opportunity Statement

How might we use campus-based ARM and digital rewards to improve convenience and motivation for returning reusable packaging, expanding participation and stabilizing return volumes?

Solution: Recycle Go Integration System

We designed an innovative circular packaging system targeting university students by establishing ARM recycling machines on campus connected to the PackAge+ network. Students can scan and return packaging at nearby return stations, earning small rewards or points. After cleaning, packaging re-enters the usage cycle through reverse logistics, creating a closed-loop system that increases return rates while reducing costs.

Key Research Insights

Behavioral Barriers

  • 75% drop-off rate: Inconvenience and weak incentives lead to high abandonment
  • Distance sensitivity: Participation significantly decreases beyond 500 meters
  • Time cost: Students value quick and convenient return processes

Motivation Drivers

  • Environmental responsibility: University students generally have environmental awareness
  • Economic incentives: Small rewards effectively boost participation
  • Social influence: Friend recommendations are the strongest motivator

Technical Feasibility

  • ARM System: Mature technology with reasonable deployment costs
  • PackAge+ Integration: Existing APIs support seamless integration
  • Campus Partnerships: Multiple universities expressed cooperation interest
